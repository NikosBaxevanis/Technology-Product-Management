# Technology-Product-Management
This is a Java console application developed for an MSc project

# Πληροφοριακό Σύστημα Καταστήματος Τεχνολογίας

Μια αντικειμενοστραφής εφαρμογή σε **Java** για τη διαχείριση αποθέματος, κατασκευαστών και πωλήσεων ενός ηλεκτρονικού καταστήματος τεχνολογίας (Ηλεκτρονικοί Υπολογιστές & Κινητά Τηλέφωνα).

---

## 🚀 Λειτουργικά Χαρακτηριστικά

1. **Διαχείριση Ηλεκτρονικών Υπολογιστών (PC):** Πλήρης διαχείριση (CRUD) με πεδία για Κωδικό, Λεκτικό, RAM, Αποθηκευτικό Χώρο, Κατασκευαστή, Τιμή και Ποσότητα.
2. **Διαχείριση Κινητών Τηλεφώνων (SmartPhones):** Πλήρης διαχείριση (CRUD) με πεδία για Κωδικό, Λεκτικό, Ανάλυση Κάμερας, Χρώμα, Κατασκευαστή, Τιμή και Ποσότητα.
3. **Διαχείριση Κατασκευαστών:** Καταγραφή Κωδικού, Επωνυμίας και Email, με σύνδεση των κατασκευαστών στα αντίστοιχα προϊόντα.
4. **Λειτουργίες Πωλήσεων & Στατιστικών:** * Καταχώρηση πωλήσεων σε πραγματικό χρόνο με αυτόματη χρονοσφραγίδα (`LocalDateTime`).
    * Έλεγχος επάρκειας αποθέματος πριν την ολοκλήρωση της πώλησης.
    * Εμφάνιση στατιστικών εσόδων και τεμαχίων ανά κατηγορία συσκευής και ανά κατασκευαστή.

---

## 🛠️ Τεχνικά Χαρακτηριστικά & Αρχές Σχεδίασης

* **Αντικειμενοστραφής Προγραμματισμός (OOP):** Χρήση κληρονομικότητας (κλάση `Product` ως βάση για τις `PersonalComputer` και `SmartPhone`) και πολυμορφισμού.
* **Διαχείριση στη RAM:** Όλα τα προϊόντα και οι πωλήσεις καταγράφονται σε ενιαίες συλλογές (`ArrayList`) κατά τη διάρκεια εκτέλεσης.
* **Μόνιμη Αποθήκευση (I/O Streams):** Τα δεδομένα φορτώνονται αυτόματα κατά την έναρξη (`LoadData`) και αποθηκεύονται σε αρχεία κειμένου CSV (`SaveData`) κατά την έξοδο.
* **Έλεγχος Εγκυρότητας (Validation) & Exception Handling:**
    * Αποφυγή διπλότυπων κωδικών σε προϊόντα και κατασκευαστές.
    * Έλεγχος ορίων (π.χ. απαγόρευση αρνητικών τιμών, ποσοτήτων ή hardware χαρακτηριστικών).
    * Θωράκιση της εισόδου του χρήστη με try-catch (`InputMismatchException`) ώστε η εφαρμογή να μην κρασάρει σε περίπτωση λανθασμένης πληκτρολόγησης.
* **Τεκμηρίωση κώδικα:** Πλήρης χρήση σχολίων **Javadoc** σε όλες τις μεθόδους της κεντρικής κλάσης.
